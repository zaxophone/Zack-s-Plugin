This plugin adds 4 new commands to the console.
AQUA- Runs a file in native AquaConsole code- supports plugins- Usage: cd <dir> then aqua <filename>
UPDATE- Adds a currently available release of AquaConsole- Usage: cd <aquaconsole program dir> then update <version>
NYAN- Cat
CHEESE

This plugin depends on Ionic.Zip which is distributed with the program but must be re-referred to in VS
There are no exceptions in this plugin so errors in syntax or IO will lead to crash
